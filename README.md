# Home Automation via Bluetooth

## Project Overview

This project demonstrates a Bluetooth-based Home Automation System that allows users to control household appliances using a smartphone and a Bluetooth module.

The system enables wireless communication between a mobile device and a microcontroller to switch appliances ON and OFF.

## Features

- Wireless appliance control
- Bluetooth communication
- Mobile phone integration
- Device ON/OFF switching
- Low-cost automation solution

## Components Required

- Arduino Uno / ESP32
- HC-05 Bluetooth Module
- Relay Module
- Smartphone
- Electrical Appliances

## Technologies Used

- Embedded Systems
- Bluetooth Communication
- Arduino Programming
- IoT Basics

## Working Principle

1. Smartphone sends commands via Bluetooth.
2. Bluetooth module receives commands.
3. Microcontroller processes commands.
4. Relay module controls connected appliances.
